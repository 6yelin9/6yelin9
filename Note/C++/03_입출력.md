# `C++` 03 입출력

## 출력
### 사용방법
    
``` cpp
cout << [값, 변수] << endl;
```

- `cout` : 출력문 선언
- `<<` : 앞뒤의 값을 연결
- `endl` : 다음줄로 넘어감 (End of line)
    - `'\n'` 사용해도 결과 동일 

### 예시코드

``` cpp
#include <iostream>
using namespace std;

int main()
{
    int a = 10;
    int b = 20;

    cout << a << " + " << b << " = " << a + b << endl;

    /* Output:
    10 + 20 = 30
    */

    return 0;
}
```
